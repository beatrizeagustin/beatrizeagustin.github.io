About Me


Hey! My name is Beatrize and I'm a former nurse turned front-end developer with appreciation for accessibility, animations, interactivity, and all things 90s. I enjoy abusing working with CSS and Javascript to develop web illustrations and animations. I'm always looking for new tools and solutions for strange, unconventional but fun ideas while still keeping kosher (accessible and responsive!). Along with technologies like React and Angular, my goal is to learn more tools like After Effects, Bodymovin, Lottie as well as three.js to make even more stranger things! Currently, I'm a student at Udacity's Front End Development Nanodegree program as a recipient of the Grow with Google Scholarship (2018).

tl;dr - I make things.
